# Changelog

## 0.1.4 - 2026-03-31

- added a dedicated framing section for object-position presets and raw crop positioning
- expanded the presence editor with richer coexistence presets for status, badge, and accessory combinations
- clarified the group editor with quick stack, tight, spread, and column profiles

- standardized avatar preset metadata into explicit family, archetype, variant, and size fields
- upgraded preset filtering so the avatar browser can slice the generated catalog by structured preset metadata instead of tags only

## 0.1.2 - 2026-03-31

- expanded the preset system from starter packs into larger generated button-style preset catalogs
- aligned the preset browser and preview reset flow more closely with the button-component product model
- preserved clean lint, typecheck, and production build verification after the canon sync

## 0.1.1 - 2026-03-31

- added a searchable preset catalog with full-state preset apply flow
- aligned the avatar studio more closely with the button-style UI Foundry editing model
- preserved clean lint, typecheck, and production build verification after the preset rollout

## 0.1.0 - 2026-03-30

- stabilized the standalone avatar app so build, lint, and typecheck pass
- synced shared layout and control primitives to the current UI Foundry visual standard
- kept the app fully self-contained for standalone packaging
